0Antes de empezar — Instala estos 2 programas (solo una vez)

1. Node.js

Descárgalo de nodejs.org → botón verde "LTS" → instala normalmente.

2. Python 3.10+

Descárgalo de python.org/downloads → instala.
Windows: marca "Add Python to PATH" antes de dar click en Install.

¿Ya los tienes? Verifica con node -v y python --version en tu terminal.

3Configura tus API Keys

Claude te pedirá las keys una por una. Aquí está la tabla completa:

Servicio Para qué sirve Dónde obtenerla Estado

ElevenLabs Voz del locutor en el video + música de fondo elevenlabs.io → Profile → API Keys Opcional

Sin las keys opcionales el PDF, post, historia, carrusel y email funcionan perfectamente. Solo el video y la publicación a Instagram necesitan las extras.

!Problemas comunes

"Error al generar textos"

Tu key de OpenAI no es válida o no tiene crédito. Verifica en platform.openai.com → Billing.

"El video no se genera"

Node.js no está instalado correctamente, o falta la key de ImgBB. Dile a Claude: "Revisa por qué no se genera el video."

"El servidor no inicia"

Dile a Claude Code: "Revisa por qué no inicia el servidor de Nova Realty AI y arréglalo."

Cualquier otro error

Copia el error completo y díselo a Claude Code. Él lo resuelve automáticamente.
